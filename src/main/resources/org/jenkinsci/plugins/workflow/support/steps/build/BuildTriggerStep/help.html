Triggers a new build for a given job.

Use the Pipeline Snippet Generator to generate a sample pipeline script for the build step. The snippet generator provides additional help for the build step through the help icons on the right side of the build step definition page. The additional help includes a detailed description of the wait argument and the return values available from the build step when wait is enabled.

If the wait argument is true, the return value of the build step is an object containing read-only properties related to the completed build.
